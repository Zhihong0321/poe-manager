# Path of Exile 2 Trade Manager

## Overview
A tool to manage and track Path of Exile 2 trading activities.

## Features
- **Sales Scanner**: Monitors specific stash tabs to track item listings.
- **Transaction Recorder**: Detects when items enter or leave the stash (Potential Sales/Listings) and records the history.

## Architecture
- **Language**: TypeScript (Node.js)
- **Data Storage**: SQLite (via `better-sqlite3`) for robust transaction tracking.
- **API Interaction**: Official Path of Exile API (Stash Tab endpoints).
